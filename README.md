# PromptRoll — iOS 快捷指令版

## 整体架构

```
┌─────────────────┐      ┌──────────────────┐      ┌─────────────────┐
│   快捷指令 App   │ ───▶ │   剪映/CapCut    │ ───▶ │   Safari 浏览器  │
│                 │ 复制  │   复制台词文字    │ 粘贴  │   打开提词器网页  │
│  "打开提词器"    │      │                  │      │   传递台词参数   │
└─────────────────┘      └──────────────────┘      └─────────────────┘
                                                          │
                                                          ▼
                                                  ┌─────────────────┐
                                                  │   提词器网页     │
                                                  │  (托管在服务器)   │
                                                  │  文字自动滚动     │
                                                  └─────────────────┘
```

## 使用流程

1. **复制台词**：从剪映/CapCut/备忘录复制要说的台词
2. **运行快捷指令**：点"打开提词器"
3. **粘贴台词**：快捷指令弹出对话框，粘贴台词
4. **自动打开提词器**：Safari 自动打开，台词已经在那等着了

## 快捷指令配置（手动导入）

快捷指令名称：`打开提词器`

### 步骤 1：文本输入
- 操作：`Ask for Input`
- 提示文字：`粘贴你的台词`
- 输入类型：文本

### 步骤 2：URL 编码
- 操作：`URL Encode`
- 输入：上一步的文本

### 步骤 3：打开网页
- 操作：`Open URL`
- URL：`https://teleprompter.huatc012.workers.dev/?script=【编码后的文本】`

## 提词器网页地址

部署地址：`https://teleprompter.huatc012.workers.dev`

---

## 快捷指令 JSON（可导入）

如果想直接导入，复制以下内容：

```
快捷指令名称：打开提词器
捷径URL：shortcuts://run-shortcut?name=打开提词器
```

实际需要手动在快捷指令 App 里创建，步骤见上方。
